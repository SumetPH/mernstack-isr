# Next ISR
"การพัฒนาเว็บไซต์โปรแกรมวิชา ระบบสารสนเทศทางคอมพิวเตอร์พัฒนาซอฟต์แวร์ (เทียบโอน) ด้วย จาวาสคริป เฟรมเวิร์ค (Javascript Framework) Information System."

## Technology
Front-End : Next.js  
Back-End : Node.js  
Database : MongoDB  
CSS Framework : Bulma  
Host : Heroku
